## Description

Getting things working locally

## Installation

```bash
$ npm install

```

### Running Database

```bash
$ docker compose up

```

### Setting up prisma

```bash
npx prisma init


npx prisma migrate dev --name post --preview-feature
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod


```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
